### 🔹 Linear Data Structures
- **Arrays** :- 
  1.An array is a linear data structure that stores elements in continuous memory locations.
  2.It is an index-based data structure, so elements are accessed using an index number.
  👉 Arrays can store different data types (numbers, strings, objects, etc.)

  In most languages (C, C++, Java):
  👉 Arrays store same data type only
______________________________________________________________________________________________

- **Linked Lists** (Singly, Doubly, Circular)
    1.A linked list is a collection of nodes, where each node contains data and the address of the next node.

    Singly Linked Lists :- A singly linked list is a linked list where each node contains data and the address of the next node only.

    Doubly Linked List :- A doubly linked list is a linked list where each node contains data, the address of the previous node, and the address of the next node.

    Circular Linked List :- A circular linked list is a linked list where the last node points back to the first node instead of null
______________________________________________________________________________________________
- **Stacks**
    A stack is a linear data structure that stores data in LIFO or FILO order
    (Last In First Out or First In Last Out).
    In stack, we use Push operation for adding the element and Pop operation for removing the element.
    If we try to remove an element from an empty stack, it is called stack underflow.
    If the stack limit exists and we try to add one more element, it is called stack overflow.
______________________________________________________________________________________________
- **Queues** (Simple, Circular, Priority Queue, Deque)
    Queue is also the linear data structure that stored the data like LILO or FIFO Order (Last in Last out OR First in First Out);
    In a queue, insertion of elements is done from the end, and removal of elements is done from the front.

    Queue also the some types :- 
    1️⃣ Simple Queue
    In a simple queue, elements are inserted from the end and removed from the front.

    2️⃣ Circular Queue
    In a circular queue, the last position is connected back to the first position to efficiently use memory.

    3️⃣ Priority Queue
    In a priority queue, elements are removed based on priority, not on insertion order.

    4️⃣ Deque (Double Ended Queue)
    In a deque, insertion and deletion of elements can be done from both front and end.

const matrix = [
    [1, 2, 3],
    [4, 0, 6],

];

function setMatrixZeroes(matrix) {
    const rows = matrix.length;
    const cols = matrix[0].length;

    const zeroRows = new Set();
    const zeroCols = new Set();

    // Find zero positions
    for (let i = 0; i < rows; i++) {
        for (let j = 0; j < cols; j++) {
            if (matrix[i][j] === 0) {
                zeroRows.add(i);
                zeroCols.add(j);
            }
        }
    }

    // Set rows to zero
    for (let r of zeroRows) {
        for (let j = 0; j < cols; j++) {
            matrix[r][j] = 0;
        }
    }

    // Set columns to zero
    for (let c of zeroCols) {
        for (let i = 0; i < rows; i++) {
            matrix[i][c] = 0;
        }
    }

    console.log("updated matrix :- ");
    console.log(matrix);
}

setMatrixZeroes(matrix);

    let head = null;

    function add(data) {
        if (head === null) {
            head = { data, next: null };
            return;
        }

        let temp = head;
        while (temp.next) {
            temp = temp.next;
        }

        temp.next = { data, next: null };

        let temp = head;
        let result = "";
        while (temp) {
            result += temp.data + " -> ";
