4.What is the Box Model in CSS?
    In CSS, Box Model means we can control any part of a website's content — like content, padding, border, and margin.
    These four parts help to arrange and style elements properly.
    This whole process is called the CSS Box Model.
->The Box Model in CSS is a way to describe how every HTML element is structured.
->  Content – The actual text or image
    Padding – Space around the content
    Border – Line around the padding
    Margin – Space outside the border

5.What is the difference between id and class selectors?

->id is unique — only one element in the entire HTML file can have that id.
->class can be used on multiple elements in the same HTML file.
->Use id when you want to style or access one specific element.
->Use class when many elements share the same style or behavior but can also have differences.
->If content is the same for many elements, use class.
->If content is unique for an element, use id.

8.What is the difference between visibility: hidden and display: none?

->✅ visibility: hidden
Element is hidden, but still takes up space in the layout.

Layout is not affected.

It's like the element is invisible but still "there".

Example:

.hidden-element {
  visibility: hidden;
}

👀 Result:
You can't see the element.
But the space it occupies remains.

✅ display: none
Element is completely removed from the layout.

Layout is affected — it's like the element doesn't exist at all.

Example:
.none-element {
  display: none;
}

👀 Result:
The element disappears.
No space is reserved for it.
